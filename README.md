# 贪吃蛇游戏

这是一个使用HTML、CSS和JavaScript开发的经典贪吃蛇游戏。

## 游戏功能

- 使用方向键或WASD控制蛇的移动
- 吃到食物后蛇会变长，得分增加
- 随着得分增加，游戏速度会逐渐加快
- 支持暂停/继续、重置游戏功能
- 响应式设计，适配不同屏幕尺寸
- 支持触摸屏操作

## 如何运行

1. 确保所有文件（index.html, styles.css, game.js）在同一目录下
2. 使用浏览器打开index.html文件即可开始游戏

## 游戏控制

- **方向键/WASD**: 控制蛇的移动方向
- **空格键**: 暂停/继续游戏
- **R键**: 重置游戏
- **开始按钮**: 开始新游戏或继续暂停的游戏
- **暂停按钮**: 暂停当前游戏
- **重置按钮**: 重置游戏状态

## 游戏规则

- 蛇吃到食物后得分增加10分
- 当蛇撞到墙壁或自己的身体时，游戏结束
- 每获得50分，游戏速度会稍微加快

## 技术实现

- 使用HTML5 Canvas进行游戏渲染
- 使用JavaScript实现游戏逻辑
- 使用CSS实现响应式设计和美观的界面
